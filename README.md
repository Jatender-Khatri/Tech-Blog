# Tech-Blog
A web app to manage your posts efficiently. This is my first project using jsp and servlet.
## Features
- Create, Update, View & Delete Posts
- Sign In and Sign Up
- Full Secure
## Tech Stack
## Client Side
- HTML
- CSS
- Boostrap
- JSP
## Server Side
- Java
- Servlet
## Database
- MySql
